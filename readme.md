# Gyme-System

Professional Gym Management System built using Flask, HTML, CSS and SQLite.

## Features

- User Login & Register
- Admin Dashboard
- Member Dashboard
- Trainer Dashboard
- Course Booking System
- Billing System
- Gym Schedule Management

## How to Run the Project

### 1. Download the project

Download ZIP from GitHub and extract the folder.

### 2. Open in VS Code

Open the project folder in Visual Studio Code.

### 3. Install requirements

Open terminal and run:

pip install -r requirements.txt

### 4. start the database

Run this command:

python init_db.py

### 5. Run the project

Run this command:

python app.py

### 6. Open browser

Open this link in browser:

http://127.0.0.1:5000
